《艾尔登法环》随机探索地图随机生成各种物品MOD[支持游戏最新1.03版本]
版本: 0.91

[ 下载地址 ]
https://patch.ali213.net/showpatch/187961.html#001

[ 使用方法 ]
1. 文件复制到游戏Game目录下
2. 运行launch_elden_ring_item_randomiser.exe，即可启动随机物品的游戏

[ MOD配置 ]
MOD配置文件在`ItemRandomiser/randomiserpreferences.ini`
部分配置项
1. autoequip: 是否开启自动装备捡到的物品
2. randomisekeys: 是否替换关键道具
