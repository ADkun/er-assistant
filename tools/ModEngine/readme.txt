环MOD引擎

[ 下载地址 ]
https://github.com/soulsmods/ModEngine2/releases

[ 使用说明 ]
1. 将模组要替换的文件，或regulation.bin文件放到Game/mod目录下
2. 将Mod Engine文件复制到Game目录下
3. 运行launchmod_eldenring.bat，即可以离线模式打开游戏，并自动加载regulation.bin以及模型替换MOD

[ MOD引擎说明 ]
1. MOD引擎可以在不解包，不修改原游戏文件的基础上，加载MOD进行游戏。
2. MOD引擎拥有独立的游戏启动器，因此与随机物品MOD不兼容。
3. MOD引擎拥有独立的游戏启动器，因此不需要单独关闭EAC（反作弊）。
4. MOD引擎与Mod Loader（dinput8.dll）可以同时使用。
