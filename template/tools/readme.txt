[ 说明 ]
tools/
自定义配置的工具目录，将工具文件按指定路径放入tools/下的独立文件夹中，助手会自动读取配置并执行相应的行为，对工具进行安装/卸载/运行

与mods/的配置相同，仅作目录及程序逻辑上的区分

[ 文件结构 ]
- 工具目录名/         任意工具目录名
    - config.ini     工具安装/卸载/运行相关配置
    - files.ini      指定工具文件数量，以及相对于游戏根目录Game/下的相对路径
    - files/         存放具体的工具文件，文件路径需要符号Game/下的相对路径，并配置到files.ini中

[ config.ini配置项 ]
- sName: MOD名称
- bCopy: 是否复制到游戏目录下
- bRun: 是否运行
- sRun: 运行程序的相对路径（相对于files/）
- bBak: 是否备份（仅当bCopy=true时有效）

"b"开头的项的值只能是true（开）或false（关），不区分大小写。
"s"开头的项为字符串，按说明填写。
